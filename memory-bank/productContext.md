# Product Context

## 为什么这个项目存在
在聊天群中，群友们经常玩 Steam 游戏，但不一定时刻关注 Steam 好友列表。本插件将 Steam 好友的游戏状态自动播报到群中，让群友能够实时了解彼此的游戏动态，增加互动性和趣味性。

## 解决的问题
1. **信息不同步** - 不需要打开 Steam 客户端就能知道群友在玩什么游戏
2. **游戏社交** - 看到群友在玩某款游戏时可以方便地加入
3. **游戏时间记录** - 播报游戏时长，增加社交趣味
4. **Steam 信息展示** - 查看群友的 Steam 个人主页，了解其游戏历史

## 用户体验目标
- **简洁绑定** - 用户只需发送一条命令即可绑定 Steam 账号
- **自动播报** - 无需手动操作，状态变更自动推送到群
- **美观展示** - 使用仿 Steam 原生风格的图片渲染，视觉体验好
- **轻量部署** - 不依赖无头浏览器，只用 Pillow 渲染图片

## 命令体系
| 命令 | 别名 | 说明 |
|------|------|------|
| `steamhelp` | `steam帮助` | 查看帮助 |
| `steambind [ID]` | `绑定steam` | 绑定 Steam ID |
| `steamunbind` | `解绑steam` | 解绑 Steam |
| `steaminfo [可选参数]` | `steam信息` | 查看个人主页 |
| `steamcheck` | `查看steam`, `查steam` | 查询群友状态 |
| `steamenable` | `启用steam` | 启用播报 |
| `steamdisable` | `禁用steam` | 禁用播报 |
| `steamupdate [名称] [图片]` | `更新群信息` | 更新群头像和名称 |
| `steamnickname [昵称]` | `steam昵称` | 设置玩家昵称 |

## 播报类型
- **all** - 全部播报，显示所有群友状态列表 + 文字消息
- **part** - 部分播报（默认），只显示开始游戏的玩家卡片 + 文字消息
- **none** - 仅文字消息
